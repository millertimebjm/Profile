Silvermine Nordic Snow Making

Link

Github

Description

The purpose of this Snow Making project is to provide alerts when thresholds are hit for good snow-making weather and options for future automation on-site. Using a Cellular Notecard from Blues.io, I can get more accurate on-site temperature and humidity readings, as well as provide opportunities in the future for alerts and automation related to snow-making equipment in combination with a server processing a variety of inputs.

IoT Usage

Since the weather from major API's are localized to the nearest airport to the Silvermine area, which is several miles away, more accurate temperature and humidity readings can be gathered onsite using a cellular IoT device. This local device uses a temp/humidity sensor with a solar recharged battery bank because we can't use local power.
